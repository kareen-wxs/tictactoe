# Настройка Telegram-бота

## Шаг 1: Создание бота

1. Откройте Telegram и найдите бота [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям и создайте бота
4. Сохраните полученный токен (например: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

## Шаг 2: Получение Chat ID

### Вариант 1: Через бота
1. Найдите бота [@userinfobot](https://t.me/userinfobot)
2. Отправьте ему любое сообщение
3. Бот вернет ваш Chat ID (например: `123456789`)

### Вариант 2: Через группу
1. Создайте группу в Telegram
2. Добавьте бота [@RawDataBot](https://t.me/RawDataBot)
3. В сообщении бота найдите `"id":` - это и есть Chat ID группы

## Шаг 3: Настройка .env

Откройте файл `.env` в корне проекта и добавьте:

```env
TELEGRAM_BOT_TOKEN=ваш_токен_от_BotFather
TELEGRAM_CHAT_ID=ваш_chat_id
```

## Шаг 4: Проверка

После настройки:
1. Запустите игру: `http://localhost:8000/game`
2. Выиграйте игру
3. Проверьте, что сообщение пришло в Telegram

## Примечания

- Если бот не отправляет сообщения, проверьте:
  - Правильность токена и Chat ID в `.env`
  - Что бот не заблокирован
  - Логи приложения на наличие ошибок
